ImageFlow Help

Getting Started

Open any image by double-clicking it in Finder with ImageFlow set as the viewer, dragging an image onto the app icon, or using ⌘ O from within the app. ImageFlow automatically finds all images in the same folder so you can navigate between them.

Keyboard Shortcuts

Action Shortcut
Open image… ⌘ O
Next image →
Previous image ←
Zoom in ⌘ +
Zoom out ⌘ -
Fit to screen / Reset zoom⌘ 0
Rotate left ⌘ [
Rotate right ⌘ ]
Toggle fullscreen ⌃ ⌘ F

Trackpad Gestures

Swipe left or right with two fingers to navigate between images. Pinch to zoom in and out.

Supported Formats

JPEG, PNG, HEIC/HEIF, GIF, TIFF, BMP, WebP.

Privacy Policy

Last updated: April 12, 2026

ImageFlow does not collect, store, or share any personal data. Ever.

Overview

ImageFlow ("the app") is a native macOS image viewer developed by Noe Aaron Garcia. This Privacy Policy explains how the app handles your information.

The short version: ImageFlow does not collect any data about you or your usage.

Data Collection

ImageFlow does not collect any personal information, usage data, analytics, crash reports, or any other type of data. The app has no servers, no backend, and no network connection of any kind.

File Access

ImageFlow only accesses image files that you explicitly open using the app. It reads the files you choose to view and the other images in the same folder so you can navigate between them.

The app does not modify, copy, upload, or share your files in any way. All file access happens locally on your device.

No Internet Connection

ImageFlow never connects to the internet. It works entirely offline. There are no ads, no trackers, no third-party SDKs, and no external services of any kind.

No Third-Party Sharing

Since we collect no data, we share no data with third parties. There are no analytics providers, advertising networks, or data brokers involved.

macOS Sandbox

ImageFlow runs within the macOS App Sandbox, which restricts what the app can access on your system. The app only has access to files you explicitly open through the standard macOS file picker or drag-and-drop.

Children's Privacy

ImageFlow does not collect data from anyone, including children under 13. The app is safe for all ages.

Changes to This Policy

If we ever update this privacy policy, we will update the "Last updated" date at the top of this page. Since the app collects no data, we do not anticipate any meaningful changes.
